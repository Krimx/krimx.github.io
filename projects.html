Circuit Simulator

Inspired by Sebastian Lague's series on how computer logic is handled, I created a software using Java and the JFrame library to allow the user to connect nodes to each other in order to create a logic circuit.

Tic Tac Toe

I believe that the ability to create a Tic Tac Toe bot and an environemtn to play against it in is the best way to show proficiency in a programming language. This project is my display of my skills in HTML, CSS, and Javascript.
